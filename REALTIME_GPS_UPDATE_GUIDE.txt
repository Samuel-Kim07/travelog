Travelog 실시간 GPS 추적 업데이트

교체 권장 파일:
- map.js
- index.css

변경 내용:
1. 기존 GPS 버튼을 한 번 위치 가져오기에서 실시간 위치 추적 토글로 변경
2. navigator.geolocation.watchPosition 사용
3. 이동 시 지도 위 내 위치 마커와 정확도 원이 계속 갱신됨
4. GPS 버튼이 활성 상태일 때 색상과 아이콘이 바뀜
5. GPS 시뮬레이션 시작 시 실제 GPS 추적은 자동 중지됨
6. 페이지를 벗어나면 watchPosition을 clearWatch로 정리

테스트 방법:
1. GitHub에 map.js, index.css 교체 후 Commit changes
2. 모바일에서 https://samuel-kim07.github.io/travelog/ 접속
3. 지도 오른쪽 GPS 버튼 클릭
4. 위치 권한 허용
5. 실제로 이동하거나 실내에서는 창가로 이동해 마커 이동 확인

주의:
- 실시간 GPS는 HTTPS에서 동작합니다. GitHub Pages는 HTTPS이므로 사용 가능합니다.
- 실내에서는 GPS 오차가 커질 수 있습니다.
- 배터리 절약 모드에서는 위치 갱신이 느릴 수 있습니다.
